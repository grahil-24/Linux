<h4> 2. Square Brackets </h4>
Easier way of calculating mathematical operations is using square brackets. Eg: 
	var1=$[1 + 5]
	 var2=$[ $var1 * 2 ]
	 echo var2


The bash shell mathematical operators support only integer arithmetic.
